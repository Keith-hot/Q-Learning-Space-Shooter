## 🎯 Project Overview

This project implements a **Deep Q-Network (DQN)** algorithm to train an AI-controlled spaceship in a real-time shooting game. Unlike traditional rule-based AI, our agent **autonomously learns** complex behaviors including:

- 🎯 **Predictive Aiming** - Leading shots on moving targets
- 🏃 **Evasive Maneuvering** - Dodging bullet patterns
- 🛡️ **Strategic Shielding** - Risk management with defensive skills
- ⚡ **Ultimate Timing** - Optimal use of screen-clearing abilities

**Best Performance:** 10,070 points (Episode 705, 20 waves cleared)

## 🎮 Game Features

- **Human Mode**: Control with WASD/Arrow keys, SPACE to shoot, E for shield, Q for ultimate
- **AI Mode**: Watch the trained DQN AI play automatically
- **Training Mode**: Train your own AI model from scratch
- **Skill System**: Shield (unlocks at Wave 3+), Ultimate ability (screen clear + invincibility)
- **Wave System**: 20 waves of enemies with increasing difficulty
